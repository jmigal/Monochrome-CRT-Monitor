# Monochrome CRT Monitor
A fully DIY pure-analog monochrome PAL CRT monitor containing an LM1881 sync separator, UC3843 25kV flyback driver, PLL locked horizontal deflection, EF80 pentode video amplifier, and a vertical deflection using a TDA1170D.

PROJECT STATUS: Design phase complete, BOM finalized, ready for construction pending funding.

<br>

## Why?
<br>
After ordering a copy of AoE i started scrolling through a PDF of it when i stumbled upon the television section, and that got me wondering, "Could i build one?".

Following that i quickly found out that there's hardly anything answering the question "How do i build a CRT from shits and bits?". There's a lot on how they work and how to repair them but i never found much on designing your own and i cannot put into words how infuriated i was after realising that, so im aiming to make this project's documentation answer exactly that.

Another realisation i had was that a TV would be a waste of time and effort as analog TV is dead where i live (as well as most of the globe) so i settled on a monitor, and since then i've been studying how one works, and later, designing my own, all compiled into a PDF with the reasoning behind every single component explained as well as how it works, which you can find [here](Documentation/)
