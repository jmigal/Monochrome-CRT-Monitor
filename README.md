# Scratch-Built-Monochrome-CRT-Monitor
## What is this?
A fully DIY pure-analog monochrome PAL CRT monitor containing an LM1881 sync separator, UC3843 25kV flyback driver, PLL locked horizontal deflection, EF80 pentode video amplifier, and a vertical deflection doing its best.

## Why did i make it?
This project begun around the start of May. After ordering a copy of AoE i started scrolling through a PDF of it when i stubmled upon the television section, and that got me wondering, "Could i build one?".

A TV would be a waste of effort as analog TV is dead where i live so i settled on a monitor, and since then i've been studying how one works, and later, designing my own, all compiled into a PDF with the reasoning behind every single component explained as well as how it works.
